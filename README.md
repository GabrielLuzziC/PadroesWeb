# 🎬 Valorização do Cinema Brasileiro

## 📌 Sobre o Projeto
Este projeto tem como objetivo principal promover, divulgar e valorizar o cinema nacional e sua rica diversidade cultural. A plataforma foi idealizada para facilitar o acesso a informações detalhadas sobre produções brasileiras de todas as regiões do país, organizando sinopses, elencos, diretores e avaliações em um só lugar.

O projeto foi desenvolvido no âmbito da disciplina de Padrões Web (ICSM49) do curso de Engenharia de Computação da UTFPR.

## 🚀 Funcionalidades Principais
* **Catálogo Completo:** Listagem organizada de filmes com sistema de filtros por gênero, ano e região.
* **Mapa Interativo do Brasil:** Uma ferramenta geográfica que permite aos usuários explorar a produção cinematográfica específica de cada região (Norte, Nordeste, Centro-Oeste, Sudeste e Sul).
* **Sistema de Recomendações:** Sugestões personalizadas baseadas nas preferências do usuário, como categorias, gêneros e diretores favoritos, facilitando a descoberta de novas obras.
* **Página de Destaques:** Uma vitrine atualizada com os filmes mais bem avaliados, últimos lançamentos ou obras indicadas a prêmios.

## 🔄 Metodologia
O gerenciamento das tarefas é feito utilizando o modelo **Kanban**. A escolha se deu pela flexibilidade com prazos, visualização clara do fluxo contínuo e facilidade em adaptar a rotação de papéis entre os membros, promovendo aprendizado contínuo e colaboração.
